
# Data Platform Portfolio

This repository contains practical analytics and data engineering work
focused on SQL analysis, Power BI reporting, and small-scale data pipeline examples.

The goal of this portfolio is to demonstrate how business questions are translated
into data queries, reports, and simple data workflows.

Repository Structure

01-sql-analysis
Business-driven SQL queries and analysis projects using relational datasets.

02-powerbi-reports
Interactive dashboards built in Power BI using modeled datasets.

03-data-pipelines
Examples of simple data movement and orchestration workflows using Azure tools.

Tools Used

SQL Server
WideWorldImporters
AdventureWorks
Power BI
Azure Data Factory
