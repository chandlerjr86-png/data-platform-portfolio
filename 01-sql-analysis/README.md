# SQL Analysis

This folder contains business-oriented SQL analysis examples built using
the WideWorldImporters dataset.

Each script represents a small analytical workflow where a business
question evolves into progressively deeper SQL queries.

The goal is to demonstrate practical SQL skills used in analytics roles,
including joins, aggregation, filtering, and business-rule logic.

Files

customer-order-analysis.sql  
Analyzes customer ordering behavior by counting orders and calculating
total revenue generated per customer.

inventory-risk-analysis.sql  
Examines warehouse inventory levels and classifies products into
operational risk categories such as OUT OF STOCK, CRITICAL, and LOW.
